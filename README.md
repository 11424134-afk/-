# 半無限大固體暫態熱傳導分析 (Transient Heat Conduction in Semi-Infinite Solid)

本專案針對鋼材在淬火工藝中的暫態熱傳導問題進行理論解析與數據化分析。

## 1. 問題描述
將初始溫度均勻為 $T_0 = 20^\circ\text{C}$ 的厚鋼板（近似為半無限大固體），在 $t = 0$ 時刻將表面浸入溫度 $T_s = 800^\circ\text{C}$ 的淬火鹽浴中。假設表面溫度暫態達到鹽浴溫度，求解鋼板內部不同位置 $x$ 處的溫度隨時間的變化規律，並計算 $t = 100\text{ s}$ 時，深度 $x = 0.05\text{ m}$ 處的溫度。

* **鋼板熱擴散率 ($\alpha$)**：
  $$\alpha = 1.5 \times 10^{-5} \text{ m}^2/\text{s}$$
  --## 2. 數學模型與解析解
一維暫態熱傳導控制方程式為：
$$\frac{\partial T}{\partial t} = \alpha \frac{\partial^2 T}{\partial x^2}$$

考慮第一類邊界條件，其理論解析解（高斯誤差函數 $\text{erf}$）為：
$$T(x,t) = T_s + (T_0 - T_s) \cdot \text{erf}\left(\frac{x}{2\sqrt{\alpha t}}\right)$$

代入本題已知參數後，溫度場分佈公式為：
$$T(x,t) = 800 - 780 \cdot \text{erf}\left(\frac{x}{2\sqrt{1.5 \times 10^{-5} \cdot t}}\right)$$

---

## 3. 特定點計算結果 ($t = 100\text{s}, x = 0.05\text{m}$)
1. **計算無因次位置參數 ($z$)**：
   $$z = \frac{0.05}{2 \times \sqrt{1.5 \times 10^{-5} \times 100}} = \frac{0.05}{0.07746} \approx 0.6455$$
2. **查表/計算誤差函數值**：
   $$\text{erf}(0.6455) \approx 0.6387$$
3. **求解溫度**：
   $$T(0.05, 100) = 800 - 780 \times 0.6387 = 301.81^\circ\text{C}$$
